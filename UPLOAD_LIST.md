## File Upload Instructions

1. **Create an `images` folder** in your website directory:
   ```
   /Users/jameschang/personal website/HungYangChang.github.io/images/
   ```

2. **Upload photos** to the `images` folder with the exact filenames listed above

3. **Photo specifications**:
   - **Format**: JPEG or PNG
   - **Size**: Optimized for web (max 500KB per image)
   - **Dimensions**: 
     - Profile photos: 400x400 pixels (square)
     - Horizontal photos: 1200x800 pixels
     - Vertical photos: 800x1200 pixels
